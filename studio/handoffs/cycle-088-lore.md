# Cycle 88 — Lore Handoff

**Theme:** The ritual, seen. Cycle 87 taught a lone dino to invent a small private
ritual (405) and cycle 88's next-up finally lets the *keeper* stumble onto one mid-act —
so the tic stops being a thing only the dead air holds and becomes a thing the player can
catch. This is a "living minds" cycle: the same private moment should read differently
depending on who the dino is and how it feels about being seen.

**Suggested next-up:** **BACKLOG-408 [social] Caught mid-tic** — greet a dino while it's
deep in its signature ritual (405) and it startles (😳), its next line coming out bashful,
as if caught doing something private. It's the queued tic follow-up that's most *player-facing*
(the whole point of the tic was distinctness the player notices) and it slots cleanly onto
the existing tone-greet path (142) — the player, not being a dino, never counts as the
company that breaks a tic, so a dino can genuinely be mid-ritual when you walk up. Deterministic
bashful framing (model-optional), so it tests without a brain.
